# Ghi chú Markdown cơ bản

## Một số cú pháp Markdown

- **Tiêu đề**: Dùng dấu `#` ở đầu dòng
- **In đậm**: Dùng `**chữ**`
- **In nghiêng**: Dùng `*chữ*`
- **Danh sách**: Dùng `-` hoặc `1.`
- **Đường link**: `[Tên](URL)`
- **Gạch ngang**: `---`
- **Code**: Dùng dấu \`code\`
